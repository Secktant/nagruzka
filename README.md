# Нагрузка

Личный калькулятор будущей финансовой нагрузки — PWA без бэкенда.
Периоды (15-е и конец месяца), регулярные платежи, рассрочки, графики нагрузки.

- **Локально, приватно:** все данные хранятся в браузере (IndexedDB), на сервер ничего не уходит.
- **PWA:** ставится на экран Домой (iPhone/Mac), работает офлайн.
- **Без подписок и аккаунтов.**

## Запуск локально

```sh
cd app && python3 -m http.server 4323
# открыть http://127.0.0.1:4323
```

## Данные

Свежая установка стартует с демо-данных (`js/seed.js`). Свои данные —
через **Настройки → Импорт** (формат — экспорт из этого же приложения).
Личные данные в репозиторий не входят.

## Стек

Ванильный JS (ES-модули), без сборки и зависимостей. Графики — инлайн-SVG.
